# Personal-Portfolio
Modern, responsive developer portfolio showcasing full-stack projects and skills. Built with vanilla HTML, CSS &amp; JavaScript.

# Features

🎨 Modern Design - Clean, professional interface with smooth animations
🌓 Dark/Light Mode - Toggle between themes with persistent preference
📱 Fully Responsive - Optimized for mobile, tablet, and desktop
⚡ Fast & Lightweight - Vanilla JavaScript, no heavy frameworks
🎯 Interactive UI - Smooth scrolling, hover effects, and animations
💼 Project Showcase - Display your best work with descriptions and tech stacks
📧 Direct Contact - Gmail integration for easy communication
🔄 Dynamic Typing Effect - Animated name and title display

# Technologies Used

# Frontend:
HTML5
CSS3
Vanilla JavaScript

# Features:
